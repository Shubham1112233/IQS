Our Services

NDT Testing

Non-destructive testing (NDT) is a wide group of analysis techniques used industry to evaluate the properties of a material, component or system without causing damage.

Liquid Penetration Testing

Liquid penetrant testing is a widely applied method used to locate surface-breaking defects in all non-porous materials (metals, plastics, or ceramics).

Magnetic Particle Testing

Magnetic particle testing is a non-destructive testing method for detecting surface and slightly subsurface discontinuities in ferromagnetic materials.

Eddy Current Testing

Eddy current testing is a non-destructive testing method that uses electromagnetic induction to detect flaws in conductive materials.

Stationary MPI Machine Testing

It is a non-destructive testing method used to detect surface and slightly subsurface discontinuities in ferromagnetic materials.

Crack Part Testing

It is a non-destructive testing technique used to identify and evaluate surface and subsurface cracks in materials.
